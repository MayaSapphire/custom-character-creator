Instructions for importing this loadout:
1. Copy the output: select inside the text box, then hit CTRL+A followed by CTRL+C.
2. Paste the output into Blahshroom's DMs.
3. Press the + button next to the message, then hit "upload your message as a file".
4. Add the command "loadout import", hit Enter, and hope that it works.
